TP SO - Delibird

El trabajo consistió en hacer un programa en C donde aplicar los conocimientos de la materia Sistemas Operativos. Esta compuesto por un planificador, una parte de manejo de memoria y otra parte el File System. Personalmente estuve involucrado en el desarrollo del planificador, donde tuvimos que implementer algoritmos FIFO, Round Robin y SJF Con y Sin Desalojo. Para esto fue necesario el uso de hilos y semaforos.

Algoritmo de Thompson

El algoritmo de Thompson sirve para obtener autómatas finitos no deterministas con transiciones vacías a partir de una expresion regular definida. Valida si una palabra cumple con una expresion regular, representada en una matriz. Desarrollado en C para la materia Sintaxis y Semantica de Lenguajes, UTN.
